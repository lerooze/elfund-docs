Κανόνες μετατροπής IFDAT σε DAT
===============================

Κάθε συμπληρωμένος πίνακας IFDAT μετατρέπεται σε έναν η περισσότερους πίνακες DAT.

Στις εγγραφές όλων των πινάκων θα προστεθούν τα εξής πεδία:

#. SRC_ORG: Το αναγνωριστικό (username) του χρήστη που έκανε την υποβολή.
#. SRC_URR: To ID του οργανισμού που έκανε την υποβολή (πχ. '030')
#. SRC_TYP: Παίρνει την τιμή 'OTR' όταν τα στοιχεια προέρχονται από τον πίνακα REF.CNTRPRTY και την τιμή 'SDR' για τα στοιχεία που προέρχονται από όλους τους άλλους πίνακες πλην του  REF.CNTRPRTY.
#. TIMESTAMP: Χρονοσφραγίδα εισαγωγής των στοιχείων στη βάση.

Για τα παρακάτω όπου αντλούνται στοιχεία από την βάση BED στο WHERE προστίθονται τα SRC_ORG, SRC_USR, SRC_TYP και λαμβάνουν τις παραπάνω τιμές. Επίσης αντλείται η πιο πρόσφατη εκδοχή με βάση το TIMESTAMP (latest version).  Αν όμως αναφέρεται ότι αντλούνται οι AUTH παρατηρήσεις από τη BED τότε αντλείται το latest AUTHORITATIVE RECORD όπως εξηγείται στις προδιαγραφές της BED.
